# Videos Folder

Place your two background videos here with these exact names:

- `video1.mp4`  → Left panel video
- `video2.mp4`  → Right panel video

Recommended:
- Format: MP4 (H.264)
- Resolution: 1920×1080 or higher
- Duration: 10–30 seconds (looping)
- Keep file size under 15 MB each for fast loading
